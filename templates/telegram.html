How to Use @KrishiBot

Get instant AI-powered assistance for farming, finance, and governance with @KrishiBot on Telegram, available in English, Hindi, and Kannada.

Find @KrishiBot

Search for @KrishiBot in Telegram or click the link below.

Find @KrishiBot

Open Telegram, tap the search icon, and type @KrishiBot. Select the bot from the results.

Start the Chat

Tap Start or type /start to begin.

Start the Chat

Once you find @KrishiBot, press the Start button or send /start to get a welcome message.

Choose Assistant

Select Financial, Legal, or Veterinary Assistant and a language.

Choose Assistant

Pick an assistant type (e.g., Financial) and your preferred language (English, Hindi, or Kannada) from the menu.

Ask Questions

Ask about loans, land disputes, or livestock health.

Ask Questions

Type questions like "How to get a farm loan?" or "What if my cow has a fever?" for quick, clear answers.
